Overview

We approximate the definite integral \[ \int_0^1 x^2 \, dx = \frac{1}{3} \] using the trapezoidal rule. Below, you can select different numbers of subintervals \( n \) to observe how the approximation improves.

Code Used

    
The following JavaScript function implements the trapezoidal rule:

    
function trapezoidalRule(f, a, b, n) {
  const h = (b - a) / n;
  let sum = 0.5 * (f(a) + f(b));
  for (let i = 1; i < n; i++) {
    sum += f(a + i * h);
  }
  return sum * h;
}

    
This is applied to the function \( f(x) = x^2 \) over the interval \([0, 1]\).
