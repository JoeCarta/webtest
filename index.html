Overview

We approximate the integral \[ \int_0^1 x^2 \, dx = \frac{1}{3} \] using the trapezoidal rule. Choose a value of \( n \) (number of subintervals) below to see how accuracy improves: 


  


  

    
The Code

    
This is the JavaScript function implementing the trapezoidal rule:

    
function trapezoidalRule(f, a, b, n) {
  const h = (b - a) / n;
  let sum = 0.5 * (f(a) + f(b));
  for (let i = 1; i < n; i++) {
    sum += f(a + i * h);
  }
  return sum * h;
}
